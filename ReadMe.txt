Программа находит самую длинную возрастающую цепочку простых чисел в бинарном файле, используя алгоритм Миллера Рабина.

Первым аргументом командной строки указывается путь к бинарному файлу или его имя ( BinaryFile )
В программе есть генератор бинарных файлов двумя разными методами, в зависимости от второго аргумента:
 -generateRandom сгенерирует массив на 60000000 ( LengthOfFile ) случайных 6-ти байтовых чисел
 -generate сгенерирует один из проверочных массивов заданных чисел ( закомментированны в Program.cs ).

Бинарный файл трактуется как последовательность 6-ти байтовых беззнаковых целых. Размер файла может быть любым, если размер файла не кратен 6, то лишние байты с конца файла игнорируются. Элементы цепочки не обязаны идти друг за другом в файле, между элементами цепочки могут встречаться не простые числа в любом количестве. Из двух цепочек одинаковой длины предпочтение отдается той, у которой первый элемент больше. Если длины и первые элементы цепочек совпадают, предпочтение отдается той, у которой смещение первого элемента меньше.

Файл с данными указывается как параметр командной строке. Реализация многопоточна. Во время работы отображается прогресс обработки файла.

Процесс выполнения прерывается клавишей ESC.

QuickEdit Mode консоли может приостанавливать выполнение процесса при клике на окно до нажатия любой клавиши, рекомендую отключать его на время выполнения.

Как результат работы выводится первый элемент цепочки и его смещение, последний элемент цепочки и его смещение и длина цепочки, или, если цепочка не была найдена – сообщение об этом прискорбном факте. Если работа приложения была прервана, то выводится текущее состояние (информацию по найденной цепочке или сообщение о том, что ничего пока не нашлось).


Примеры последовательностей и найденные цепочки:
Последовательность --- Найденная цепочка
2, 3, 5, 3, 5, 7, 11  ---  3, 5, 7, 11
2, 3, 4, 5, 6, 7, 3, 5, 7 --- 2, 3, 5, 7
2, 3, 10, 5, 6, 7, 3, 5, 7 --- 2, 3, 5, 7
2, 3, 3, 5, 6, 7, 3, 5, 7 --- 3, 5, 7
2, 3, 5, 3, 5, 11 --- 3, 5, 11
2, 3, 5, 2, 5, 11 --- 2, 3, 5
